AIFRED VST3 Free Beta 0.1.0
===========================

Free public beta.

Active in this beta:
- Core analyzer
- Compare Mode
- Local Reference Mode for up to five user-loaded tracks inside the plugin
- Local AI route through Ollama at http://127.0.0.1:11434 using aifred:latest. Installed packages may also start the AIFRED gateway at http://127.0.0.1:8787 for plugin health/chat routing.
- Optional OpenAI-compatible routing
- Website Reference Intake for free limited analysis and curated reference-pool candidates

Important limits:
- Cloud reference-profile sync is roadmap/flagship unless a release explicitly says it has been verified.
- Website uploads receive limited analysis and pass through the quality gate before accepted metadata helps build the curated pool.
- The plugin uses local user-loaded reference tracks for beta Reference Mode; it does not need the website for that.
- AIFRED does not mix the song for you. It interprets measured audio and gives practical guidance.

Install notes:
1. Download the beta artifact for your platform from the GitHub beta release.
2. Run the installer or copy the VST3 according to the included platform notes.
3. Run the included local AI setup script if the installer reports Ollama, aifred:latest, or AifredEngine is missing.
4. Load AIFRED in your DAW and rescan VST3 plugins if needed.
