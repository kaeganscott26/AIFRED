AIFRED v0.3.3 Ollama Chat Release Notes

Current build target:
- JUCE VST3 plugin with resizable Halo interface.
- Halo quadrant scale ticks, reference rings, and switchable center display.
- Ten persisted session candlesticks plus 10 one-minute history candlesticks.
- Reference mode with five reference volume lanes.
- Compare mode with separate Mix A and Mix B analysis routes.
- $5 PayPal purchase path with tokenized backend download delivery.
- Local Ollama chat using current DSP context and reference-pool metadata.
- Optional OpenAI-compatible endpoint setup when the user supplies private credentials.
- Website activity records now persist to the admin log feed and notify the Android app for buy clicks, order events, catalog plays, inquiries, downloads, and admin uploads.
- Catalog audio uploads auto-sync the repo catalog JSON plus the audio file so the website and admin app stay aligned.
- VST chat now shows real DSP values in the chat panel and uses a larger scrollable response area.
